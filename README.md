# Генератор задач ЕГЭ по математике

Веб-приложение для генерации задач ЕГЭ по математике (Задание №16 - Проценты и вклады) без использования нейросетей.

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui компоненты
- React Router
- TanStack Query

## Установка и запуск

### Локальная разработка

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для production
npm run build

# Предпросмотр production сборки
npm run preview
```

### GitHub Actions

Проект настроен для автоматической сборки и деплоя через GitHub Actions:

1. При пуше в ветки `main` или `master` автоматически запускается сборка и деплой на GitHub Pages
2. При создании Pull Request запускается проверка кода и сборка

## Структура проекта

```
src/
├── components/          # React компоненты
│   ├── pages/          # Страницы приложения
│   ├── ui/             # UI компоненты (shadcn/ui)
│   ├── lib/            # Утилиты и генератор задач
│   └── hooks/          # React хуки
├── index.css           # Глобальные стили
└── main.tsx            # Точка входа
```

## Особенности

- Алгоритмическая генерация задач без использования нейросетей
- Поддержка различных шаблонов задач (найти сумму, начальную сумму, процентную ставку)
- Три уровня сложности (легкий, средний, сложный)
- Пошаговые решения задач
- Адаптивный дизайн
